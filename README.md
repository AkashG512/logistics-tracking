# FastTrack Logistics

**World Class Delivery. Logistics made Simple.**

FastTrack Logistics is a modern, responsive web application designed to streamline shipment tracking and management. It provides a seamless experience for customers to track their packages and a secure admin panel for logistics managers to update shipment statuses.

## Getting Started

### Prerequisites

You need a modern web browser (Chrome, Firefox, Safari, Edge) to run this application. No backend server or installation is required as it is a client-side application.

### Installation

1.  Clone the repository or download the source code.
2.  Navigate to the project directory.
3.  Open `index.html` in your web browser.

## Usage

### Tracking a Shipment

1.  Go to the **Track Shipment** page.
2.  Enter one of the demo Tracking IDs (e.g., `TRK001`, `TRK002`).
3.  Click **Search** to view the current status.
